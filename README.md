SkillSphere is a modular system that aggregates GitHub data to automatically generate an interactive 3D knowledge graph representing a student's skill progression.

## Functional Scope

- GitHub data ingestion
- Skill extraction
- Skill categorization (taxonomy-based)
- Relationship detection between skills
- Graph generation
- 3D visualization
- Search & filtering
- Manual refresh for periodic updates

## Architecture Overview

1. GitHub API Service
2. Skill Extraction Engine
3. Taxonomy Classification Layer
4. Relationship Builder
5. Graph Construction Module
6. 3D Visualization Frontend

## Tech Stack

Backend:
- Node.js
- Express

Frontend:
- React
- react-force-graph-3d
